Lesson 9 — Next Steps

Wrap-up~5 min

You just did a real assessment in miniature. Here's where to go from here.

📚 The full benchmarks

  • CIS Ubuntu Linux 24.04 LTS Benchmark v1.0.0 — free with email signup. ~250 controls; we covered ~12.
  • DISA STIG Ubuntu 24.04 (PDF, hosted with this site) — DoD's stricter cousin. Reading the V-IDs you missed is a fast way to find your weak spots.

🛠 Automate this audit

  • OpenSCAP / oscap — official open-source SCAP scanner. Runs CIS or STIG profiles against a host and emits HTML reports. Quick start: 
    sudo apt install -y libopenscap8 ssg-debderived
sudo oscap xccdf eval --profile cis_level1_server \
  --results /tmp/results.xml --report /tmp/report.html \
  /usr/share/xml/scap/ssg/content/ssg-ubuntu2404-ds.xml
  • Lynis — lightweight, no datastreams needed: sudo lynis audit system. Best for "give me a quick second opinion."
  • CIS-CAT Pro — CIS's official scanner, members-only.
  • Wazuh — open-source SIEM with built-in CIS policy monitoring across many hosts.

🎓 Where to learn more about defensive assessments

  • SANS SEC511 — Continuous Monitoring & Security Ops. The blue-team gold standard.
  • GIAC GMON / GCED — paired certifications.
  • OffSec OSDA — hands-on defensive analyst cert.
  • CompTIA CySA+ — entry-level blue-team cert; good vocabulary builder.
  • LFCS / RHCSA — Linux depth that pays for itself in every assessment.

🔭 What a real engagement looks like

This course was a taste. A real engagement covers the full ~250 controls of the benchmark, requires evidence collection (screenshots, command output, file hashes) for every finding, includes attestations from the system owners on accepted risks, and re-tests after remediation. The hard part isn't running the commands — it's the conversation with the customer about which findings they'll fix vs. accept, the documentation discipline that makes the report reproducible six months later, and the scoping conversation where you decide what's in and out before you ever ssh in.

Most assessor tooling can be learned in a weekend. The judgment to use it well is what gets paid for.
